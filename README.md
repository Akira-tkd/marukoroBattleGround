# **作品概要**
  **作品名:**まるころBattleGround

  **ジャンル:**三人称アクションシューティング

  **制作期間:**3ヶ月

  **制作人数:**:人

  **概要:**
  
  ウェーブ制の三人称シューティングゲームです。

  敵を全滅させると次のウェーブに進行し、各ウェーブ終了時に強化要素の選択肢がランダムで出現します。

  プレイヤーは4種類のキャラクターが選択可能で、敵の出現パターンや強化内容が毎回ランダムなため、状況に応じたプレイが求められます。

# **使用技術**
  ・Unity
  ・C#
  ・ScriptableObject
  ・Interface設計
  ・StateMachine

# **制作工程**
  1. 三人称視点のプレイヤーキャラクターとカメラの実装
  2. Input Systemのアクションマップによる操作制御の実装
  3. ScriptableObjectを用いた攻撃データ、キャラクターデータ管理の設計
  4. 攻撃の追加効果を拡張可能にするための基底クラスの作成
  5. Interfaceを用いた攻撃処理の抽象化
  6. StateMachineによるゲーム進行管理の実装
  7. キャラクター選択機能、敵のランダム出現システムの実装


# **工夫点**
  **・拡張性を重視**
    ScriptableObjectとInterfaceを用いることで、既存コードを変更せずに攻撃や敵を追加可能に

  **・高速対応の当たり判定**
    弾の当たり判定にRayを使用し、弾やプレイヤーなどが高速移動時でも安定したヒット判定を実現

  **・段階的チュートリアル**
    操作が段階的に解放されるチュートリアルを実装することで、プレイヤーが自然にゲームに慣れる設計に
